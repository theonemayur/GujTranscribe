# GujTranscribe - Standalone Version

## How to Use

1. Double-click `Run_GujTranscribe.bat` to start the application
2. Wait a few seconds for the server to start
3. Your browser will open automatically at http://localhost:8000
4. That's it! No installation needed!

## System Requirements

- Windows 10 or later
- Internet connection (for first run - downloads AI models)
- 8GB+ RAM recommended

## Files Included

- `GujTranscribe.exe` - The main application
- `gujarati_asr/` - Required files (keep this folder)
- `Run_GujTranscribe.bat` - Launcher script

## Features

- 6 Whisper AI models for transcription
- Gujarati speech-to-text
- Subtitle generation (SRT)
- Vocabulary management
- Batch transcription
- Voice recording
- Dark/Light theme

## Troubleshooting

If the app doesn't start:
1. Make sure `GujTranscribe.exe` and `gujarati_asr` folder are in the same location
2. Right-click `Run_GujTranscribe.bat` and select "Run as administrator"
3. Check if port 8000 is available
